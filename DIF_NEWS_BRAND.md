# 🎯 DiF News - Brand Identity

## O Que É DiF News?

**DiF News** é o boletim semanal minimalista e sofisticado para profissionais do ecossistema contábil e financeiro.

Chega toda terça-feira às 9:00 AM (Brasília) com as notícias mais importantes da semana.

---

## 🎨 Identidade Visual

### Marca
```
DiF News

Tipografia: Sistema (sans-serif)
Tamanho: Pequeno, elegante
Estilo: Uppercase, letter-spacing
Cor: #1a1a1a (preto sofisticado)
```

### Paleta
```
Ouro Sofisticado    #D4AF37  (10-15% do design)
Preto              #1a1a1a  (texto principal)
Cinza Escuro       #666     (corpo)
Cinza Médio        #999     (metadata)
Cinza Claro        #bbb     (labels)
Branco             #ffffff  (fundo)
Cinza BG           #fafafa  (fundo secundário)
```

### Princípios
```
✨ Menos é Mais
- Apenas #D4AF37 em detalhes estratégicos
- Zero emojis no layout principal
- Uma única fonte sistema

🎭 Profissionalismo
- Weight 300 para leveza
- Letter-spacing para refinamento
- Border-radius 2px (quase reto)

📖 Legibilidade
- Line-height 1.6-1.8
- Espaçamento generoso (32px, 48px)
- Max-width 600px

⚡ Velocidade
- Font sistema (sem web fonts)
- CSS minimalista
- HTML limpo
```

## 📧 Email Template

### Estrutura
```
┌────────────────────────────┐
│       HEADER               │  Header com accent line
├────────────────────────────┤
│       CONTENT              │  Artigos numerados
│                            │  (01, 02, 03...)
│       CTA                  │  Botão elegante

│       FOOTER               │  Links + copyright
└────────────────────────────┘
```

### Numeração
```
01  Primeiro Artigo
02  Segundo Artigo
03  Terceiro Artigo
...

Estilo: Font-weight 300, #D4AF37, 20px
Guia a leitura naturalmente
```

### Links
```
Cor: #D4AF37
Estilo: UPPERCASE + arrow (LER ARTIGO →)
Peso: 600, letter-spacing 1px
Sem underline
```

### Botão CTA
```
Texto: "Explorar Todas as Notícias"
Fundo: #D4AF37
Texto: Preto (#000)
Hover: opacity 0.9
Border-radius: 2px
Padding: 14px 32px
```

## 📱 Implementação Técnica

### Email HTML
- Font sistema (não web fonts)
- Responsive (600px max)
- Válido W3C
- 10-15KB (muito leve)

### Espaçamento
- Header: 48px padding
- Content: 48px padding
- Footer: 32px padding
- Entre artigos: 32px

### Cores
- 4 cores no total
- #D4AF37 em 10-15% do espaço
- Sem gradientes
- Sem efeitos de sombra

### Tipografia
- Nenhuma web font
- Apenas system fonts
- Weights: 300, 400, 600
- Sizes: 11px a 32px
