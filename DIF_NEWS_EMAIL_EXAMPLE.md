## 🎨 Detalhes de Design

```
DiF News           ← 12px, UPPERCASE, letter-spacing 1.2px, #999
Destaques da Semana ← 32px, weight 300, letter-spacing -0.5px, #1a1a1a
━━━━━━             ← 2px line, #D4AF37 (40px wide, centered)
```

### Artigos
```
01  Selic em alta...  ← Número (20px, weight 300, #D4AF37)
```

## 📱 Visualização Responsiva

### Desktop (600px)
```

```

### Mobile (320px)
```
Tudo redimensiona automaticamente
- Padding lateral: 20px
- Números continuam 20px (grandes o suficiente)
- Texto continua legível
- Flex layout se adapta
```

## 🎯 Características Principais

### ✨ Sofisticado
- Tipografia refinada (weight 300 para leveza)
- Espaçamento generoso
- Apenas 1 cor de accent (#D4AF37)
- Sem elementos gráficos pesados

### 🎪 Minimalista
- Sem emojis no layout
- Sem imagens de fundo
- Sem cores flutuantes
- Sem efeitos visuais

### 📖 Legível
- Hierarquia clara (números guiam)
- Corpo em #666 (não preto puro)
- Line-height 1.6-1.8 (respiração)
- Max-width 600px (readability)

### ⚡ Rápido
- Font sistema (não web fonts)
- CSS minimalista
- Sem imagens pesadas
- Código limpo

## 🎬 Fluxo de Interação

1. **Chegada do Email** (Terça 9AM)
   - Subject: "DiF News - Destaques da Semana"
   - De: news@dif.com.br
   - Design reconhecível

2. **Leitura**
   - Usuário escaneia números (01, 02, 03...)
   - Títulos chamam atenção (weight 600)
   - Preview convida a clicar

3. **Ação**
   - Clica em "LER ARTIGO →" (cor #D4AF37)
   - Vai ao artigo completo no portal
   - Pode compartilhar

4. **Preferências**
   - Clica em "Preferências" (no footer)
   - Gerencia frequência ou segmentos
   - Nunca sai forçado
